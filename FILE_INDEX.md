## 🔍 Code Organization

### Modules (Rust's way of organizing code)

```
autonomi_rust_backend
├── main                    (src/main.rs)
├── config                  (src/config.rs)
├── models                  (src/models/mod.rs)
├── services
│   └── chunk_service      (src/services/chunk_service.rs)
└── handlers
    └── chunk_handler      (src/handlers/chunk_handler.rs)
```

### Dependencies (in Cargo.toml)

**Web Framework:**
- `actix-web` - Like FastAPI
- `actix-cors` - CORS middleware

**Async Runtime:**
- `tokio` - Like Python's asyncio

**HTTP Client:**
- `reqwest` - Like httpx

**Serialization:**
- `serde` - Like Pydantic
- `serde_json` - JSON support

**Utilities:**
- `base64` - Base64 encoding
- `dotenv` - .env file loading
- `log` + `env_logger` - Logging

**Testing:**
- `tokio-test` - Async test support
- `mockito` - HTTP mocking
